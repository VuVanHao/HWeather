# HWeather
 1. Mô tả : 
 + HWeather là một ứng dụng dự báo thời tiết trên thiết bị Android.
 + Ứng dụng cung cấp thông tin chi tiết về thành phố vào từng thời gian trong ngày và liên tục trong 5 ngày.
